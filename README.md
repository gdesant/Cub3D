## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/gdesant/Cub3D.git
~~~

Go to the project directory  

~~~bash  
  cd Cub3D
~~~

Compile 

~~~bash  
make all
~~~

Start the game

~~~bash  
./Cub3D
~~~
